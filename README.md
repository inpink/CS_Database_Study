## How to proceed 📖
1. `2주 동안 3단원 분량을 각자 공부`
    1.  `글로 정리하여 개인 블로그에 업로드`
    2. `각 단원별로 “예상 질문지와 답변을 5개 이상씩” 준비`
2. `랜덤으로 각자 한 개의 단원을 배정`
3. `각자 맡은 단원을 정리한 내용을 바탕으로 설명, 발표`
4. `예상 질문지 45개를 각자 하나씩 랜덤하게 뽑고, 글이나 책을 보지 않고 면접하듯 답변`
